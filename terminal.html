Welcome to Ronnie Keating's Portfolio! This is a command line-like interface using HTML, CSS, and JavaScript.

Type 'help' (without quotes) for a list of commands or type 'resume' to view my resume.

Currently available commands:

clear: clears screen. Also takes the argument 'history' to clear history.

echo: outputs whatever text proceeds it.

ls: lists the files in the directory (very cheesy, not legit at all).

resume: redirects to a pdf of my resume.

history: prints out command history.
